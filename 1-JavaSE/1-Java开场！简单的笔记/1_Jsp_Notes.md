# 1-Java开场！简单的笔记

## 开场语
学编程写代码，需要有语言语法的基础，但是最重要的是需要有编程思维。
如何编如何开发，需要从需求分析到系统设计，进行流程标准化

## jdk与jre
- **jdk**：java开发工具包，包含jre等
- **jre**：java运行环境，包含java虚拟机（jvm）等。

## Java语言特点
- **面向对象**：java是一种面向对象的编程语言，所有的代码都是在类中实现的。
- **跨平台**：java程序编译后会生成.class二进制字节码文件，此文件被目标系统识别，即可运行。
- **支持多线程**：java程序可以同时执行多个线程，提高程序的运行效率。

## Java核心机制
### JVM
Java虚拟机。java文件编译后会生成.class二进制字节码文件，此文件被目标系统识别，即可运行
### GC
垃圾回收机制。java程序运行时，会分配内存空间，当内存空间不再被使用时，自动及时回收，避免内存泄漏。
java程序进行内存管理时，分为栈内存和堆内存，栈内存存放基本数据类型，引用数据类型，堆内存存放对象实例。
在C/C++中，内存管理是malloc，在堆内存中申请空间，然后栈中声明指针指向对应的堆内存分配的空间。在释放内存时，需要程序员手动操作。而Java是自动的。
